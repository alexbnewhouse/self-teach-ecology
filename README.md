# Open-Source Self-Taught Master's in Conservation Ecology and Rewilding

An open, rigorous, self-directed curriculum designed for building master's-level expertise in conservation ecology and rewilding — benchmarked against UC Davis, Yale, Duke, ETH Zurich, and Imperial.

> **Tailored for**: A learner with a PhD-in-progress in Political Science, an MS in Data Science and Analytics, strong Python/R/ML/network analysis skills, and limited formal ecology training. Adaptable to other backgrounds with Module 0 adjustments.

---

## Quick Start

1. **Check your foundations** → Start with [Module 0: Prerequisites](curriculum/00-prerequisites.md) to self-assess biology and earth science gaps
2. **Pick a study pace** → See [Study Tempo](#study-tempo) below
3. **Track your progress** → Use [PROGRESS.md](PROGRESS.md) as your living checklist
4. **Run practice exercises** → `python3 exercises/auto-graded/autograde.py list-banks` ([exercise docs](exercises/README.md))
5. **Understand mastery targets** → Review the [Competency Framework](resources/competency-framework.md)

## Curriculum Map

The curriculum follows a tiered progression. Complete each tier before advancing to the next (parallelization opportunities are noted within modules).

### Tier 0: Bridge and Prerequisites

| Module | Hours | Prerequisites | Assessment | Status |
|--------|-------|---------------|------------|--------|
| [**Module 0**: Prerequisites & Self-Assessment](curriculum/00-prerequisites.md) | 10–30 | None | Self-audit + reflection | &#9744; |

### Tier 1: Core Ecology and Conservation

| Module | Hours | Prerequisites | Parallel With | Assessment | Status |
|--------|-------|---------------|---------------|------------|--------|
| [**Module 1**: Foundations of Ecology](curriculum/01-foundations-of-ecology.md) | 80–100 | Module 0 | — | Reading synthesis + notebooks + essay | &#9744; |
| [**Module 2**: Conservation Biology](curriculum/02-conservation-biology.md) | 80–100 | Module 1 | — | Applied analyses + synthesis essay | &#9744; |
| [**Module 3**: Rewilding Theory & Practice](curriculum/03-rewilding-theory-practice.md) | 60–80 | Module 2 | Modules 4–5 | Case analysis + monitoring design + essay | &#9744; |
| [**Module 4**: Landscape Ecology & Spatial Analysis](curriculum/04-landscape-ecology-spatial-analysis.md) | 60–80 | Modules 1–2 | Modules 3, 5 | Spatial notebooks + conservation planning report | &#9744; |
| [**Module 5**: Quantitative Methods for Ecology](curriculum/05-quantitative-ecology.md) | 60–80 | Module 1 + DS background | Modules 3–4 | Reproducible notebooks + quantitative synthesis | &#9744; |
| [**Module 6**: Conservation Genetics](curriculum/06-conservation-genetics.md) | 40–60 | Modules 1–2 + Module 0 genetics | Modules 3–5, 7 | Simulation notebook + genetics management brief | &#9744; |
| [**Module 7**: Climate Change Ecology](curriculum/07-climate-change-ecology.md) | 40–60 | Modules 1–2 | Modules 3–6 | Range-shift analysis + climate adaptation plan | &#9744; |
| [**Module 8**: Conservation Policy & Governance](curriculum/08-conservation-policy-governance.md) | 60–80 | Modules 2–3 + PSCI background | Modules 4–7 | Policy analyses + comparative governance synthesis | &#9744; |

**Tier 1 total**: ~480–640 hours

> **Parallelization guide**: After completing Modules 1–2, Modules 3–8 open up with significant overlap. Modules 3, 4, and 5 can run concurrently. Modules 6 and 7 can parallel most others. Module 8 requires Module 3 but can otherwise overlap with 4–7.

### Milestone: Comprehensive Exam

| Assessment | Hours | Prerequisites |
|------------|-------|---------------|
| [**Comprehensive Exam**](assessments/comprehensive-exam-blueprint.md) | 10–16 | Modules 1–8 complete |

Four written essays (1,000–1,500 words each) covering ecological theory, conservation governance, quantitative methods, and an integrative applied scenario. Validates master's-level synthesis before the capstone.

### Tier 2: Specialization Tracks (Choose 2–3)

| Track | Hours | Prerequisites | Focus |
|-------|-------|---------------|-------|
| [Systems Ecology & Complexity](curriculum/tracks/systems-ecology-complexity.md) | 45–70 | Modules 1, 4, 5 | Feedbacks, resilience, regime shifts, network ecology |
| [Restoration Ecology (Advanced)](curriculum/tracks/restoration-ecology-advanced.md) | 45–70 | Modules 1, 2, 4 | SER standards, succession, soil ecology, adaptive monitoring |
| [Computational Ecology](curriculum/tracks/computational-ecology.md) | 80–120 | Modules 4, 5 (rec: 7) | End-to-end pipelines: SDM, Bayesian, remote sensing, networks, ML/AI for conservation |
| [Political Ecology](curriculum/tracks/political-ecology.md) | 50–75 | Modules 2, 3, 8 | Canon, conservation-as-politics, restoration governance, quantitative political ecology |

**Tier 2 total**: 90–190 hours (2 tracks minimum)

Recommended bundles:
- **Systems & Forecasting** → Systems + Computational (regime shifts, network-driven capstones)
- **PSCI → Data/ML Ecologist** → Political Ecology + Computational with Pathway F (your optimal path)
- **Policy-Relevant Restoration** → Restoration + Political Ecology (governance research)
- **Applied Rewilding Analytics** → Computational + wildlife/marine domain (data-heavy evaluations)

Additional elective domains (marine, wildlife, freshwater, conservation tech) are documented in the [Electives Index](curriculum/09-electives.md).

### Tier 3: Capstone Research Project

| Module | Hours | Prerequisites | Formats |
|--------|-------|---------------|---------|
| [**Module 10**: Capstone Research Project](curriculum/10-capstone-project.md) | 100–180 | Modules 1–8 + 2–3 tracks + exam draft | Empirical research, systematic review, policy-ecology report, or computational pipeline |

Produces a near-publishable research artifact. 20-week phased workflow: proposal → methods → analysis → write & defend.

### Tier 4: Applied Product Development

| Module | Hours | Prerequisites | Output |
|--------|-------|---------------|--------|
| [**Module 12**: Business of Rewilding](curriculum/12-business-of-rewilding.md) | 40–60 | Modules 1–8 + 1 track + capstone in progress | Market landscape analysis + revenue models + case studies |
| [**Module 13**: Rewilding Product — Concept to MVP](curriculum/13-rewilding-product-development.md) | 120–200 | Modules 1–10 + Module 12 | Working prototype + user testing + launch plan |

Tier 4 bridges ecological expertise into a consumer-facing rewilding product — native plant guidance, biodiversity tracking, smart garden systems, or community rewilding platforms. Designed for public good with passive income potential.

## Assessment and Mastery

### Competency Domains

Your target is **Proficient** or above across all domains by capstone completion.

| Domain | Modules | Target |
|--------|---------|--------|
| Ecological Theory | 1, 2, 4, 7 | Proficient → Advanced |
| Quantitative & Computational Methods | 4, 5, 6 + Computational Track | Advanced (given DS background) |
| Conservation Decision & Governance | 2, 3, 8 + Political Ecology Track | Advanced (given PSCI background) |
| Field & Applied Practice | 3, 4 + Restoration Track + Practicum | Proficient |
| Scholarly Synthesis & Communication | All modules + Exam + Capstone | Advanced |
| Applied Product Development | 12, 13 | Proficient (Tier 4) |

Full framework: [resources/competency-framework.md](resources/competency-framework.md)

### Assessment Milestones

| Milestone | When | What |
|-----------|------|------|
| Module self-assessments | End of each module | Exercises, notebooks, essays per module rubric |
| Auto-graded exercises | Throughout | `python3 exercises/auto-graded/autograde.py run --bank <bank> --learner <name>` |
| Comprehensive exam | After Tier 1 | 4 synthesis essays ([blueprint](assessments/comprehensive-exam-blueprint.md)) |
| Track deliverables | Tier 2 | Per-track projects and written syntheses |
| Capstone | Tier 3 | Near-publishable research artifact ([rubrics](assessments/rubrics.md)) |
| Product MVP | Tier 4 | Working prototype + user testing report |

### Portfolio Artifact Checklist

- [ ] 6+ reproducible computational notebooks
- [ ] 4+ substantial written analyses (2,000+ words each)
- [ ] 1+ restoration or monitoring design document
- [ ] Comprehensive exam essay responses
- [ ] Capstone project repository with full reproducibility artifacts
- [ ] Market landscape analysis + product concept document (Tier 4)
- [ ] Working MVP with user testing documentation (Tier 4)

## Benchmark Alignment

This curriculum is designed to be equivalent in depth and rigor to ecology master's programs at:

| Program | Key Equivalences |
|---------|-----------------|
| **UC Davis** (ECL 200A/200B) | Modules 1, 4, 7 (core ecology); 2, 3 (conservation emphasis); 5, 6 (methods) |
| **Yale** (MEM/MESc) | Modules 1, 2, 8 (foundations); 5 (methods); Tier 2 tracks (specialization); Module 10 (thesis) |
| **Duke** (MEM) | Modules 1, 2, 4, 7 (environment); 5 + Computational Track (analytics) |
| **ETH Zurich** (Ecology & Evolution) | Modules 1, 6, 7 (foundation); 3, 4, 5 (advanced); GIS, Bayesian, SDM, ABM, networks |
| **Imperial** (Conservation Science) | Module 0 (bridge); 8 + Systems Track (systems thinking); Restoration practicum (applied) |

Full mapping: [curriculum/11-benchmark-mapping.md](curriculum/11-benchmark-mapping.md)
