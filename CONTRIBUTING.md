# Contributing

Thanks for improving this curriculum.

## Contribution Types

- Resource improvements (replace outdated or lower-quality sources)
- Citation corrections
- Module clarity improvements
- New exercises or assessment prompts

## Resource Inclusion Criteria

1. High scholarly quality and direct relevance
2. Preference for open-access materials when equivalent
3. Clear placement in module or track sequence
4. Concise rationale for why it improves the curriculum

## Citation Style

- Include complete author, year, title, venue, and DOI/URL when available
- Avoid link-only citations without bibliographic detail

## Pull Request Expectations

- Describe what changed and why
- Link to any issue or discussion context
- Keep edits scoped and avoid unrelated formatting churn
- Update navigation links if adding new files

## Versioning

The curriculum uses semantic releases:

- Major: substantial structural or philosophy changes
- Minor: added tracks/modules/assessments
- Patch: citation fixes, links, typo corrections
