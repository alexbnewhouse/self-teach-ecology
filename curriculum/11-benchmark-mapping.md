# Module 11: Benchmark Mapping to Leading Ecology Master's Programs

**Estimated time**: 6–10 hours  
**Type**: Audit and synthesis module  
**Prerequisites**: Familiarity with Modules 0–10

---

## Purpose

This module documents how the curriculum aligns with master's-level expectations from:

- UC Davis Graduate Group in Ecology
- Yale School of the Environment (MEM/MESc)
- Duke Nicholas School (MEM)
- ETH Zurich (Environmental Sciences, Ecology & Evolution major)
- Imperial College (Conservation Science / Living Planet)

---

## Universal Pillars Coverage Check

| Pillar | Benchmark Expectation | Curriculum Coverage |
|---|---|---|
| Core ecology theory | 1–2 graduate ecology cores | Modules 1, 2, 4, 7 |
| Quantitative methods | At least one graduate methods sequence | Modules 4, 5, 6 + Computational Track (incl. Pathway F: ML/AI) |
| Research methods/design | Formal methods training and seminar practice | Module 0 research prep, Module 10 methods phases, assessments blueprint |
| Thesis/capstone | Independent project with substantial written output | Module 10 capstone + assessment rubrics |
| Social-environmental theory | Governance, institutions, human dimensions | Module 8 + Political Ecology Track |

---

## Program-by-Program Mapping

## UC Davis (ECL 200A/200B + emphasis)

- **Core ecology series equivalent**: Module 1 (foundations) + Module 4 (landscape) + Module 7 (climate ecology)
- **Conservation emphasis equivalent**: Module 2 + Module 3 + elective tracks
- **Methods readiness**: Module 5 quantitative ecology, Module 6 genetics, capstone methods section

## Yale MEM/MESc (foundations + methods + specialization + thesis)

- **Foundations and perspectives**: Modules 1, 2, 8
- **Methods requirement equivalent**: Module 5 + Assessments comprehensive blueprint
- **Specialization credits equivalent**: Tier 2 tracks (choose 2–3; Political Ecology track aligns with Yale’s environmental social science theory requirement)
- **Thesis credits equivalent**: Module 10 4-phase capstone workflow

## Duke MEM (environment + analytics)

- **Environment concentration equivalent**: Modules 1, 2, 4, 7
- **Analytics/modeling concentration equivalent**: Module 5 + Computational Ecology Track
- **Dual-concentration integration**: Module 8 governance + data-intensive capstone options

## ETH Zurich Ecology & Evolution major

- **Foundation block**: Modules 1, 6, 7
- **Advanced concepts**: Modules 3, 4, 5 + Systems Ecology and Restoration tracks
- **Technical competencies**: GIS/remote sensing, Bayesian modeling, SDM, ABM, network analysis
- **Seminar/term paper equivalent**: module syntheses + comprehensive exam essays
- **Thesis equivalent**: Module 10 capstone deliverable

## Imperial Conservation Science / Living Planet

- **Bridge for non-biologists**: Module 0 prerequisite pathway
- **Systems thinking and decision focus**: Module 8 + Systems Ecology Track
- **Applied methods and practical work**: Landscape, quantitative, restoration practicum, field checklists
- **Social science / environmental anthropology**: Political Ecology Track
- **Conservation technology / ML**: Computational Ecology Pathway F
- **Independent project**: Module 10 capstone with reproducibility and quality checks

## Gaps and Ongoing Improvements

1. Add more explicit organismal and field-methods prerequisites in Module 0.
2. Keep annual updates for methods software stacks and package recommendations.
3. Add optional mentor-reviewed oral defense protocol for capstone completion.
4. Political Ecology track extends governance coverage beyond most MS programs — a unique strength for the PSCI-background learner.
5. ML/AI Pathway F goes beyond typical MS methods requirements; positions learner for conservation data science roles.

## Note on Tier 4

Modules 12–13 (Applied Product Development) extend beyond standard master's program benchmarks into conservation entrepreneurship and applied product design. This tier is unique to this curriculum and reflects the learner's goal of creating a personal rewilding product that bridges ecological expertise with public-facing tools.
