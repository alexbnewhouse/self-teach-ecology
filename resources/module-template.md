# Standard Module Template

Use this template for all current and future modules/tracks.

## Metadata Block (Required)

```markdown
**Type**: Core module / specialization track / assessment
**Estimated time**: X–Y hours
**Prerequisites**: Module(s)
**Assessment**: Quiz / notebook / essay / project / synthesis
**Benchmark mapping**: UC Davis | Yale | Duke | ETH | Imperial
```

## Recommended Structure

1. Purpose
2. Learning objectives (5–7 specific outcomes)
3. Core resources (1 primary + optional alternates)
4. Practice tasks
5. Deliverables and evaluation criteria
6. Synthesis prompt
7. Next-step links

## Resource Metadata (Per Resource)

Include:

- Title
- Resource type (textbook, paper, MOOC, lecture, dataset)
- URL or DOI
- Estimated effort
- Why this resource is included

## Assessment Metadata (Per Exercise)

Include:

- Skill focus
- Inputs/data required
- Expected output
- Time estimate
- Rubric mapping

## Quality Guardrails

- Prefer open-access sources when quality is equivalent
- Keep scope curated (avoid long undifferentiated lists)
- State assumptions and limitations in every project prompt
- Require reproducibility for all computational work
